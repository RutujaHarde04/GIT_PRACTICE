Git and Github practice

GitHub is a popular platform for version control and collaboration, primarily used by developers. It is built around Git, an open-source version control system that helps track changes in code and allows multiple people to work on the same project simultaneously.

    Basic features and concepts of GitHub:
  1. Repositories (Repos)
  2. Commits
  3. Branches
  4. Pull Requests
  5. Forking
